# GashaPMS — Gasha Placement Management System

A web-based platform for Gasha Institute graduates to discover, apply for, and manage job placements — powered by PHP, MySQL, HTML, and CSS.

---

## 🎯 Overview

**GashaPMS** is a **Placement Management System** designed to help final-year students and recent graduates of Gasha Institute connect with job opportunities. It automates job listings, application tracking, and news updates — all in one centralized, easy-to-use portal.

---

## ✨ Key Features

- **Job Listings & Applications** — Browse and apply for jobs directly through the platform
- **Live News Feed** — Automatically fetches and displays the latest job postings and announcements from Gasha's official news page
- **User Dashboard** — Track application status, view history, and manage profile
- **Authentication System** — Secure login and registration for students and admins
- **Admin Dashboard** — Automatic redirection to admin panel for administrators upon login
- **Database-Driven** — Secure backend powered by MySQL for user data, applications, and job records
- **Responsive UI** — Clean, modern design built with HTML & CSS for all devices

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3
- **Backend:** PHP 7.4+
- **Database:** MySQL
- **Architecture:** MVC-inspired (if applicable)
- **Deployment:** Apache/Nginx + PHP-FPM

- **XAMPP** (Apache, PHP, MySQL bundled) — [Download here](https://www.apachefriends.org/)
- A code editor (VS Code, Sublime Text, etc.)

1. **Install XAMPP:**

   - Start Apache and MySQL services from the XAMPP Control Panel
